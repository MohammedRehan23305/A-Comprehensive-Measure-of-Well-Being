# HDI Predictor — Flask + Pickle ML App

Predict a country's **Human Development Index (HDI)** using a Linear Regression
model serialised with Python's `pickle` module and served via Flask.

---

## Project Structure

```
hdi_project/
├── app.py              ← Flask web application
├── train_model.py      ← Model training & pickle save
├── HDI.pkl             ← Saved model (generated by train_model.py)
├── requirements.txt
├── templates/
│   └── index.html      ← Prediction UI
└── static/
    └── style.css       ← Styles
```

---

## Quick Start

### 1 · Install dependencies
```bash
pip install -r requirements.txt
```

### 2 · Train the model & save HDI.pkl
```bash
python train_model.py
```
This trains a `LinearRegression` model on synthetic HDI data and saves it:
```python
# saving our model into a file
pickle.dump(reg, open('HDI.pkl', 'wb'))
```

### 3 · Run the Flask app
```bash
python app.py
```
Open **http://127.0.0.1:5000** in your browser.

---

## How Pickle Works

| Step | Code | Description |
|---|---|---|
| **Save** | `pickle.dump(reg, open('HDI.pkl','wb'))` | Serialises the trained model to a byte stream on disk |
| **Load** | `pickle.load(open('HDI.pkl','rb'))` | Deserialises the model back into a usable Python object |

Saving the model eliminates the need to retrain it on every application start,
saving time and computational resources.
