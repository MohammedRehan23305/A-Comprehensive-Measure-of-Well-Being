About the Model

This application uses a Linear Regression model trained on synthetic HDI data. The trained model is serialised as HDI.pkl via Python's pickle module and loaded at Flask startup — no retraining required.

How Pickle Saves & Loads the Model

1
Train

A LinearRegression model is fitted on the training data inside train_model.py.

2
Serialise (Save)

pickle.dump(reg, open('HDI.pkl', 'wb')) converts the model object to a byte stream and writes it to disk.

3
Deserialise (Load)

pickle.load(open('HDI.pkl', 'rb')) in app.py restores the model — ready to predict instantly, no retraining.

4
Predict

User inputs are forwarded to model.predict() and the HDI value is displayed in real time.
